Chorus Bot - Full Dependency Inventory

This document lists all libraries and prerequisites used across the project (frontend, serverless API, and Python translation service), including required tooling.

============================================================
Core Tooling / Prerequisites
============================================================
- Node.js: >= 18.x (Next.js 14 requirement)
- npm: >= 9.x (or pnpm/yarn equivalent)
- Python: >= 3.8 (virtualenv recommended)
- Git: latest stable
- Vercel account (for deployment)
- Optional (GPU acceleration):
  - NVIDIA GPU with CUDA toolkit + cuDNN compatible with installed PyTorch
- OS utilities (used by scripts):
  - Bash/sh (for install/start scripts)
  - wget or curl (for model/script downloads; optional if not using script)

============================================================
Serverless API (Next.js API Route)
============================================================
- Uses Next.js built-in serverless functions
- Calls Google Cloud Translation v2 via HTTPS
- Fallback: MyMemory free tier (English → Indic) when Google is unavailable
- Required env variable:
  - GOOGLE_TRANSLATE_API_KEY (Google Cloud API key with Translation scope)

============================================================
Legacy Python Translation Service (IndicTrans2 via Hugging Face)
============================================================
Purpose: Historical local/hosted service to perform Indic translations with the HF model `ai4bharat/indictrans2-en-indic-1B`. No longer required for standard deployments.

Python Packages (see chorus-bot/requirements.txt):
Core ML Stack
- torch: >= 2.0.0
- torchvision: >= 0.15.0
- torchaudio: >= 2.0.0

Transformers & Inference
- transformers: >= 4.39.0
- accelerate: >= 0.28.0
- tokenizers: >= 0.15.0
- sentencepiece: >= 0.1.99
- safetensors: >= 0.4.2

NLP & Utilities
- sacremoses: >= 0.0.53
- indic-nlp-library: >= 0.91
- mosestokenizer: >= 2.0.1
- scikit-learn: >= 1.3.0
- pandas: >= 2.0.0
- pyarrow: >= 12.0.0
- sacrebleu: >= 2.3.0
- tensorboardX: >= 2.6.0
- submitit: >= 1.4.0

Web Service
- flask: >= 2.3.0
- flask-cors: >= 4.0.0
- gunicorn: >= 21.2.0 (optional; for production hosting)

Notes:
- Model: ai4bharat/indictrans2-en-indic-1B (downloaded automatically via HF hub on first use)
- trust_remote_code=True is used to enable src_lang/tgt_lang kwargs for the pipeline
- GPU acceleration is optional but recommended for large workloads
- Treat this stack as optional/legacy; Google Cloud Translation is the primary path
